# pereklada4
Cайт для полегшення вивчення англіської мови, група ІПЗ-13 "Пільмені".
Склад команди та ролі:
- Єфремов Валерій — Team Lead
- Манзик Іван — Frontend developer
- Гаврилюк Назар  — Backend developer
- Степанюк Віктор  — Designer 
- Зубик Дмитрій — Documenter

Назва проєкту: QuapsE

Проблема: Багато студентів стикаються з труднощами при вивченні англійської мови, особливо під час перекладу та запам'ятовування нових слів. Сайт допоможе покращити знання англійської та зробить навчання інтерактивним.

Користувачі: Студенти, школярі та всі, хто бажає покращити рівень англійської мови.

Очікуваний результат: Створення простого та зручного веб-додатку-перекладача з функціями тренувань, який допоможе користувачам ефективно вивчати англійську.

Інструменти для реалізації:
JavaScript, React.js, Node.js, Express, CSS (UI), VS Code (IDE).

-----ІНСТРУКЦІЯ: Запуск юніт-тестів Translator e проєкті-----

1️. Переконайся, що в проєкті є файли тестів, наприклад:
   src/components/Translator.test.js

2️. Відкрий термінал у корені проєкту (там, де package.json).

3️. Запуск тестів:
   npm test
   або один раз без режиму перегляду:
   npm test -- --watchAll=false

4️. Перевір результат:
   PASS  src/components/Translator.test.js
   ✓ рендериться базово
   ✓ виконує переклад через LibreTranslate
   ✓ відображає помилку, якщо обидва сервіси не відповіли
   ✓ змінює напрямок перекладу

✅ Якщо всі пункти позначені PASS — все працює правильно. Ти молодець!

-----ІНСТРУКЦІЯ: запуску проєкту:-----
 1. Переконайся, що у проєкті є всі необхідні файли (src, public, node_modules, package.json, backend, images тощо).
 2. Відкрий термінал у кореневій теці проєкту (де знаходиться package.json).
 3. Уведи команду npm run dev і зачекай кілька секунд, поки проєкт запуститься у браузері.
 4. Проєкт відкриється за адресою http://localhost:3000

Також можна запустити онлайн за посиланням: https://cernusoknatalia-ops.github.io/pereklada4/
Сайт доступний як на комп’ютері, так і на телефоні.

Додатково можна запустити через мобільний застосунок (APK):
 1. Завантаж файл QuapsE.apk на свій смартфон.
 2. Дозволь установку додатків із невідомих джерел (за потреби).
 3. Відкрий застосунок QuapsE — функціонал повністю ідентичний вебверсії.

Короткий посібник користувача(User Manual):
Після запуску або переходу на сайт QuapsE відкривається головна сторінка з можливістю авторизації.
На сайті є такі основні розділи:
 1. Авторизація / Профіль — увійди або створи акаунт, щоб зберігати свій прогрес і проходити тести.
 2. Перекладач — перекладай слова, фрази або речення між українською та англійською мовами.
 3. Словник — містить уживані слова з перекладом і транскрипцією, можна додавати власні.
 4. Граматика — розділ із правилами, прикладами й таблицями для кращого розуміння.
 5. Тести — перевір свої знання та збережи результати у профілі.

Додаткові можливості:
 • Перемикач світлої/темної теми.
 • Адаптивний дизайн для ПК і телефону.
 • Синхронізація даних між вебверсією та APK-додатком.

Висновок:
QuapsE — це сучасний навчальний вебсервіс і мобільний застосунок для вивчення англійської мови. Він об’єднує перекладач, словник, граматику, тести й авторизацію, забезпечуючи комфортне та ефективне навчання
